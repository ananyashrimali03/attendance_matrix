How to Use

  1. Upload your ORIGINAL Excel file - the one with all the attendance sheets from Google Forms
  2. Your Excel file should have:
    • One main sheet with student roster (names and emails)
    • Multiple sheets with attendance data (one per class session)
  3. Each attendance sheet needs an Email Address column
  4. The roster sheet should have:
    • A column for student names (Student, Name, etc.)
    • A column for student emails (Email, Andrew, Email Address, etc.)
  5. Click "Generate Matrix" and the app will:
    • Count total classes (number of attendance sheets)
    • Count how many classes each student attended
    • Calculate attendance percentages
Important: Upload your original Excel file with attendance data, NOT a file you've already downloaded from this app!
